# ⚡ Pokémon Clicker Edition ⚡

Ein webbasiertes Incremental / Clicker-Game, bei dem du durch Klicken Level aufsteigst, ein starkes Team aufbaust, Upgrades kaufst und dich bis an die Spitze kämpfst! 

🎮 **[Hier direkt im Browser spielen!](https://slobidi-pixel.github.io/pokemon-clicker/)**

## 🕹️ Features
* **Click to Catch:** Sammle Punkte durch aktives Klicken.
* **Upgrades & Multiplikatoren:** Optimiere dein Team, um passive Punkte zu generieren.
* **Boss-Kämpfe:** Messe dich mit legendären Trainern (Schaffst du es bis zu Red?).
* **Responsive Design:** Direkt im Browser spielbar, keine Installation notwendig.

## 🛠️ Technologien
* HTML5
* CSS3
* Vanilla JavaScript
